### ⚠️ Important Information & Safety
- Data Source: The application uses the official public Digital Extremes API: worldState.php.
- Safety: This API is public and open to everyone. Using this application cannot lead to a ban, as it does not interact with game files in any way.
- Privacy: The application does not request or use any of your account data (login, password, email). It is fully standalone.
  
### ✨ Features
- Void Fissures Monitoring: Stay updated on active fissures in real-time.
- Desktop Notifications: Get notified about new events with custom sound alerts.
- System Tray Support: Minimize the app to the tray to keep your taskbar clean while still receiving notifications.
- Notification Management: Easily toggle specific notifications on or off and save your preferences.
