```bash
git clone https://github.com/clay1299/WarframeAlerts.git
```
### ⚠️ Важная информация и безопасность

- Источник данных: Приложение использует официальный публичный API Digital Extremes: [worldState.php](https://api.warframe.com/cdn/worldState.php?).
- Безопасность: Данный API открыт для всех, использование приложения не может привести к бану, так как оно никак не взаимодействует с файлами игры.
- Приватность: Приложение не запрашивает и не использует никакие данные вашего аккаунта (логин, пароль, почту). Оно полностью автономно.

### ✨ Функции
- Мониторинг Разломов бездны. 
- Уведомления на рабочий стол о новых событиях, со звуковым оповещением.
-  **Работа в трее:** Приложение можно свернуть в системный трей, чтобы оно не занимало место на панели задач, но продолжало присылать уведомления.
-  Сохранение уведомлений, которые можно включать и выключать
